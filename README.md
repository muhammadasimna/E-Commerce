# Lumina - Premium E-Commerce Platform

Lumina is a modern, responsive, and full-featured E-Commerce web application. It features a stunning glassmorphism design, real-time notifications for sellers, and a seamless shopping experience for buyers.

## 🌟 Key Features

-   **User Authentication**: Secure sign-up and login using JWT and Bcrypt.
-   **Seller Dashboard**: Add, edit, and delete product listings.
-   **Real-time Sales Tracking**: Sellers get notified of new orders with badges.
-   **Buyer Experience**: Browse products, view details, manage cart, and place orders.
-   **Responsive Design**: Optimized for all devices (Mobile, Tablet, Desktop).
-   **Dark/Light Mode**: Premium UI with theme switching.

```powershell

```

## 🛠️ Technology Stack

-   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
-   **Backend**: Node.js, Express.js.
-   **Database**: SQLite3.
-   **Security**: JSON Web Tokens (JWT).
